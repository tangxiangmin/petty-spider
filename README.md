
## 爬虫客户端

一个使用electron构建的爬虫客户端

* 填写抓取基本信息，包括url、页数、页面解析策略、数据保存形式等
* 点击开始按钮，等待抓取完成

### Features
* 抓取任务可视化
* 自定义页面抓取策略
